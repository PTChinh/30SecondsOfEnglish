> 30 seconds of English is a Chrome extension.

### 🏠 [Homepage](https://github.com/ngocsangyem/30SecondsOfEnglish)

## Prerequisites

-   node >=4

## Install

```sh
yarn install
```

## Usage

```sh
yarn run start
```

## Run tests

```sh
yarn run test
```
